# java-cli-millbuild-mysql-simple

## Description
A java gradle build, that connects to mysql database.

## Tech stack
- java
- millbuild
  - 6.8.2

## Docker stack
- mariadb:latest
- nightscape/scala-mill

## To run
`sudo ./install.sh -u`

## To stop
`sudo ./install.sh -d`

## For help
`sudo ./install.sh -h`
